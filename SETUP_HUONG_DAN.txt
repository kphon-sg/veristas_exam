HUONG DAN CAI DAT DU AN (CHAY MOI LAN KHI DOWNLOAD CODE)
===========================================================

Buoc 1: Mo Terminal (PowerShell hoac CMD) tai thu muc nay.

Buoc 2: Chay lenh cai dat (Bat buoc phai chay lenh nay dau tien):
npm install

Buoc 3: Doi cho den khi hien thong bao thanh cong.

Buoc 4: Chay du an Frontend:
npm run dev

Buoc 5: Chay du an Backend (Java):
- Mo Terminal PowerShell trong VS Code.
- Chay cac lenh sau:
  cd software-java
  .\mvnw.cmd spring-boot:run

-----------------------------------------------------------
Giai thich cac loi thuong gap:
1. 'vite' is not recognized: Do ban chua chay 'npm install'.
2. Cannot find module: Do ban chua chay 'npm install' hoac bi loi trong qua trinh tai.
3. Loi Database: Hay dam bao MySQL cua ban dang chay va mat khau trong file 
   'software-java/src/main/resources/application.properties' la dung.
